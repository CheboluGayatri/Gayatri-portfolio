# 📥 Getting Started

## Clone the Repository

```bash
git clone https://github.com/CheboluGayatri/Gayatri-portfolio.git
```

## Navigate to the Project

```bash
cd Gayatri-portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```
