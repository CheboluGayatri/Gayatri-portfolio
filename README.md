## 📥 Installation

Clone the repository

```bash
git clone https://github.com/CheboluGayatri/Gayatri-portfolio.git
```

Go to the project folder

```bash
cd Gayatri-portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
